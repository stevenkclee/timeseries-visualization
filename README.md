# timeseries-visualization
## Install:
```
$ git clone https://github.com/0xYTF/stocku
$ cd stocku
$ npm install
```

## Setup:
```
$ npm start
$ npm stop
```

or
```
$ node bin/www
```

default port 8000
```
$ ssh -L 8000:localhost:8000 <user@ip> -p 11222
```


## Record:
`/database`： storage for stocks 
`/public`：front-end  
`app.js`：Main program of server
`package.json`：config of npm
